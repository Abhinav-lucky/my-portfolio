# My Portfolio Website

A personal portfolio website built with HTML & CSS, 
showcasing my skills, projects, and experience as a 
DevOps Engineer Intern at DecodeLabs.

## Installation

```bash
# Clone the repository
git clone https://github.com/Abhinav-lucky/my-portfolio.git

# Navigate into the folder
cd my-portfolio
```

## Usage

```bash
# Simply open the portfolio.html file in your browser
start portfolio.html
```

## Features

- Hero Section with introduction
- About Me section
- Projects showcase
- Skills & Tools
- Internship Experience at DecodeLabs
- Education & Certificates
- Contact Information
